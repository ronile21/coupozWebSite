# CoupoZ Website

An interactive investor pitch deck website for CoupoZ - a real-time coupon and campaign management platform for physical retail businesses.

## Overview

CoupoZ empowers physical retail businesses with:
- **Real-Time Campaign Management** - Activate and stop promotions in minutes
- **Synergistic Collaborations** - Cross-promote with complementary businesses
- **Location-Based Targeting** - Reach nearby customers through geofencing and Wi-Fi positioning

## Features

- Responsive single-page application
- Interactive financial projections chart (Chart.js)
- Expandable roadmap timeline
- AI-powered campaign idea generator (Gemini API)
- Mobile-friendly navigation

## Tech Stack

- HTML5 / CSS3
- Tailwind CSS (via CDN)
- Chart.js for data visualization
- Google Gemini API for AI features
- Google Fonts (Inter)

## Getting Started

Simply open `index.html` in a web browser or deploy to any static hosting service.

For GitHub Pages deployment, the site is configured via the `CNAME` file.

## Project Structure

```
coupozWebSite/
├── index.html      # Main website
├── style.css       # Additional styles
├── CNAME           # GitHub Pages domain config
├── README.md       # This file
└── images/         # Image assets
```
